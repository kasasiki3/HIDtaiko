各項目の説明Parameters

感度

各センサーの反応閾値です。値が低いほど感度が高くなります。黄色の範囲が推奨値です。多重入力が発生する場合は値を大きくしてください。

Sensitivity

Threshold for each sensor. Lower value = higher sensitivity. The yellow zone is the recommended range. Increase the value if phantom inputs occur.

縁ディレイ

縁を連続で叩いたときに多重入力が起きる場合に大きくしてください。大きすぎると縁の連打が鈍くなります。

Rim Delay

Increase if consecutive Rim hits cause phantom inputs. Too high will slow down Rim rolls.

面ディレイ

面を連続で叩いたときに多重入力が起きる場合に大きくしてください。大きすぎると面の連打が鈍くなります。

Don Delay

Increase if consecutive Don hits cause phantom inputs. Too high will slow down Don rolls.

縁面相互ディレイ

面を叩いたときに縁が反応する、または縁を叩いたときに面が反応する場合に大きくしてください。

Cross Delay

Increase if hitting Don also triggers Rim, or vice versa.

公式/シミュレーター向け

公式アプリや一部のシミュレーターでキーが認識されない場合はONにしてください（Hdelay=20ms が適用されます）。

Official/Simulator

Turn ON if key inputs aren't recognized in the official app or some simulators (applies Hdelay = 20 ms).

キー入力後、同じキーが再び受け付けるまでの待機時間。面と縁で個別に設定できます。叩いた箇所のみで多重入力が起きる場合に値を上げてください。

Cooldown before the same sensor can fire again. Adjustable separately for Don and Rim. Increase if a single hit causes double input on the same sensor.

Cooldown after a Don (center) hit before another Don can fire. Too high will slow down rolls.

Cdelay 標準: 13 ms

Bdelay の縁バージョンです。説明は Bdelay と同様です。

Cdelay Std: 13 ms

Same as Bdelay but for Rim (edge) hits.

Ddelay 標準: 10 ms

面入力後に縁入力を制限する時間（逆も同様）。面を叩いたときに縁が多重する場合や、その逆の場合に値を上げてください。

Time to suppress Rim input after a Don hit (and vice versa). Increase if a Don hit also triggers a Rim, or vice versa.

公式アプリや一部シミュレーター向けの入力制限。「メモ帳では入力できるがシミュレーターでは入力できない」場合に設定してください。

Hdelay Std: 20 or 0 ms

Input throttle for the official app or some simulators. Set this if inputs work in Notepad but fail in a simulator.

感度調整のやり方Tuning Guide
  1. 感度スライダーを黄色の範囲内に設定する（初期値の目安）
  2. 実際に演奏して強打・弱打が確実に反応するか確認する
  3. 多重入力（1打で複数反応）が起きる場合は、該当するディレイを少しずつ大きくする
    • 縁で多重入力 → 縁ディレイを上げる
    • 面で多重入力 → 面ディレイを上げる
    • 面を叩いたとき縁が反応 → 縁面相互ディレイを上げる
  4. 調整が終わったら「感度送信」で書き込む
  1. Set sensitivity sliders within the yellow recommended range as a starting point
  2. Play and verify both strong and light hits register correctly
  3. If phantom inputs occur, slightly increase the corresponding delay:
    • Rim double input → increase Rim Delay
    • Don double input → increase Don Delay
    • Don hit also triggers Rim → increase Cross Delay
  4. When done, click "Send" to apply the settings
  1. 感度設定 強打・弱打どちらも確実に反応する中で、できるだけ値を大きく（感度を低く）設定する。
  2. Delay 調整 — 以下の順で、誤反応しない範囲でできるだけ小さい値に絞る。
    • Ddelay：面を叩いたとき縁が反応しなくなる最小値
    • Bdelay：片方の面を強打したときもう一方の面が反応しなくなる最小値
    • Adelay：同じセンサーが1打で2連発しなくなる最小値
    • Cdelay：Adelay・Bdelay と同様に縁用で調整

補足 — サイトの初期値はおおよその目安です。感度を上げると（値を小さくすると）クロストークが増え Delay を増やす必要があります。逆に感度を下げると（値を大きくすると）Delay を短くできます。

  1. Sensitivity Find the highest value (lowest sensitivity) that still reliably registers both strong and light hits.
  2. Delay tuning — In the order below, minimize each value while eliminating phantom inputs.
    • Ddelay: minimum value at which a Don hit no longer triggers a Rim
    • Bdelay: minimum value at which a hard Don hit no longer triggers the opposite Don
    • Adelay: minimum value at which a single hit no longer double-fires on the same sensor
    • Cdelay: same as Adelay / Bdelay but for Rim hits

Note — The default values on this site are rough guidelines. Raising sensitivity (lower value) increases crosstalk and requires higher delays. Lowering sensitivity (higher value) allows shorter delays.

ファームウェアのアップデートFirmware Update
準備 接続器のネジを外し、基板を取り出す
1 DFU モードで PC に接続する

マイコンボードの BOOT ボタン（左側）を押しながら USB を差し、PC と接続します。エクスプローラーにストレージとして表示されたら成功です。

2 GitHub リリースからファームウェアをダウンロードする

🔗 リリースページを開く

Assets の HIDtaiko.uf2（↓矢印で示された箇所）をクリックしてダウンロードします。

3 .uf2 ファイルを書き込む

ダウンロードした HIDtaiko.uf2 を、DFU モードで認識されたストレージへドラッグ＆ドロップします。ストレージの接続が自動的に切れたら書き込み完了です。

Prep Remove the screws and take out the PCB
1 Connect in DFU mode

Hold the BOOT button (left side) while connecting the USB cable. The device will appear as a storage drive in Explorer.

2 Download firmware from GitHub Releases

🔗 Open Releases Page

Click HIDtaiko.uf2 under Assets (indicated by the arrow) to download.

3 Flash the .uf2 file

Drag & drop the downloaded HIDtaiko.uf2 onto the DFU storage drive. The drive will disconnect automatically when done.
